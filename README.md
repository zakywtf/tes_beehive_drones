# simple-employee-system-backend-v1

REST API for simple-employee-system.

# Requiments:

- Node.JS

# Installation:

Dev Enviroments:

```cmd
$ cd simple-employee-system-backend-v1
$ npm install --save
$ npm run debug or npm run debug >> /dev/null
```

Run on

```cmd
localhost:1211
```
